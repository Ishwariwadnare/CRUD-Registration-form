# User CRUD Application

A full stack User Management CRUD Application built using Node.js, Express, MySQL, and EJS.

## 🚀 Features
- Register new user
- View all users
- Edit user details
- Delete user
- Form validation
- Success message after registration
- Clean UI with CSS

## 🛠 Tech Stack
- Node.js
- Express.js
- MySQL
- EJS
- HTML
- CSS
- JavaScript
- 
## ⚙️ Installation

Clone the repository:

```
git clone https://github.com/Ishwariwadnare/nodejs-user-crud-app.git
```

Install dependencies:

```
npm install
```

Create a `.env` file:

```
DB_HOST=localhost
DB_USER=root
DB_PASS=yourpassword
DB_NAME=formcrud
```

Run the server:

```
node index.js
```

Open browser:

```
http://localhost:3000
```
